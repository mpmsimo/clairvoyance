# clairvoyance

Drift detection and reporting for Terraform.

## Getting started

This project requires Go to be installed. On OS X with Homebrew you can just run `brew install go`.

Running it then should be as simple as:

```console
$ make
$ ./bin/clairvoyance
```

### Testing
``make test``
